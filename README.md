# Graph Neural Networks for Social Network Analysis in India: Detecting Fake Profiles & Botnets

## 📌 Overview

This project focuses on detecting fake profiles and botnets in Indian social media networks using **Graph Neural Networks (GNNs)**. Traditional machine learning models often fail to capture the complex relationships between users. By representing users and their interactions as graphs, GNNs learn structural patterns that help identify malicious accounts with higher accuracy.

---

## 🚀 Features

- Detects fake social media profiles
- Identifies coordinated botnet activities
- Graph-based social network analysis
- Uses Graph Neural Networks for classification
- High accuracy compared to traditional ML techniques
- Scalable for large social media datasets

## ⚙️ Working

1. Load the social media dataset.
2. Build a graph representing users and their interactions.
3. Extract node and edge features.
4. Train the Graph Neural Network model.
5. Detect fake profiles and botnets.
6. Evaluate model performance using classification metrics.
